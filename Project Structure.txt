EXECUTION FLOW (what happens when you run: python main.py)

   main.py
     ├─→ src/data_loader.py           (Load X, y)
     ├─→ src/splitter.py              (Train/Val/Test split)
     ├─→ src/preprocessing.py         (Build ColumnTransformer)
     ├─→ src/cv_pipeline.py           (Wrap with SMOTE)
     ├─→ src/models.py                (Define 4 classifiers)
     ├─→ src/tuning.py                (Optuna search → best_params)
     ├─→ src/cross_validate.py        (5-fold CV on all models)
     ├─→ src/save_model.py            (Fit + save best model → models/best_model.pkl)
     ├─→ src/threshold.py             (Find τ* on validation set)
     ├─→ src/evaluate.py              (Eval on test set → metrics)
     ├─→ src/visualize.py             (Generate all plots → figures/)
     └─→ src/shap_analysis.py         (SHAP explanations → figures/)
